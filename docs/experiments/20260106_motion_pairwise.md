# Motion埋め込みのペア比較（同/異ラベル）

- 対象run: `20260104_hoyo_m2t_repro`
- 入力: `hoyo_v1_1/joint_training_results/20260104_hoyo_m2t_repro/latent_snapshot_final.npz`
- 出力: `hoyo_v1_1/joint_training_results/20260104_hoyo_m2t_repro/motion_pairwise_report.json`

---

## 1) 何を見たか

- motion埋め込み（z_m）同士のcos類似度を全ペアで比較
- 同ラベル同士 vs 異ラベル同士の分布で分離度を評価
- ラベル間平均類似度の表で混同ペアを特定
- Silhouette score（cos距離）でクラスタ分離の指標を算出

---

| 中央値 | 0.695 | 0.490 | 0.206 |
| 分布の重なり | 同↓(<=異med): 0.13 | 異↑(>=同med): 0.19 |  |

- Silhouette score（平均）: **-0.114**

## 5) 解釈

- 同ラベルと異ラベルの平均差が小さい場合、スタイル分離が弱い
- 分布の重なりが大きいと、報酬差が出にくい
- 混同しやすいペアを優先的に評価・デバッグすると効率的
